# Minecraft Mod MCP について

Minecraft Mod MCP は**モデルコンテキストプロトコル（MCP）**を通じて、実行中のゲームを AI エージェントに公開します。

AI は画面を検査し、GUI をクリックし、コマンドを実行できます。Mod のテスト、ブロックやアイテムの動作検証、反復作業の自動化に最適です。すべてのコマンド（クリック・入力・スクリーンショット）は **Java リフレクション**で動くため、単一コードベースで対応する全バージョンに対応し、バージョンごとのパッチは不要です。
