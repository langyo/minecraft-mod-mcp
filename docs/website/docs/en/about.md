# About Minecraft Mod MCP

Minecraft Mod MCP exposes the running game to AI agents via the **Model Context Protocol**.

Your AI inspects screens, clicks through GUIs, and runs commands — perfect for testing mods, verifying block and item behavior, and automating repetitive workflows. Every command (click, type, screenshot) works through **Java reflection**, so a single codebase runs across every supported version — no version-specific patches.
