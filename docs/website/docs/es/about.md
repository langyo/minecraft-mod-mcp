# Acerca de Minecraft Mod MCP

Minecraft Mod MCP expone el juego en ejecución a los agentes de IA mediante el **Model Context Protocol**.

Tu IA inspecciona pantallas, hace clic en GUIs y ejecuta comandos: ideal para probar mods, verificar el comportamiento de bloques y objetos, y automatizar flujos repetitivos. Cada comando (clic, escribir, captura) funciona con **reflexión Java**, por lo que una única base de código sirve para todas las versiones compatibles, sin parches por versión.

**[Cómo empezar →](https://github.com/langyo/minecraft-mod-mcp#getting-started)**
