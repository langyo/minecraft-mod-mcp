# О Minecraft Mod MCP

Minecraft Mod MCP открывает запущенную игру ИИ-агентам через **Model Context Protocol**.

Ваш ИИ проверяет экраны, кликает по интерфейсам и выполняет команды — идеально для тестирования модов, проверки поведения блоков и предметов, а также автоматизации повторяющихся задач. Каждая команда (клик, ввод, скриншот) работает через **рефлексию Java**, поэтому единая кодовая база поддерживает все совместимые версии без правок для каждой версии.
