# À propos de Minecraft Mod MCP

Minecraft Mod MCP expose le jeu en cours d'exécution aux agents IA via le **Model Context Protocol**.

Votre IA inspecte les écrans, clique dans les interfaces et exécute des commandes : idéal pour tester des mods, vérifier le comportement des blocs et objets, et automatiser les tâches répétitives. Chaque commande (clic, saisie, capture) fonctionne par **réflexion Java**, donc une seule base de code couvre toutes les versions prises en charge, sans correctifs par version.
