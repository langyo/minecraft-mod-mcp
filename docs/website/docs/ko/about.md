# Minecraft Mod MCP 소개

Minecraft Mod MCP는 **모델 컨텍스트 프로토콜(MCP)**을 통해 실행 중인 게임을 AI 에이전트에 노출합니다.

AI가 화면을 검사하고, GUI를 클릭하고, 명령을 실행할 수 있습니다. 모드 테스트, 블록과 아이템 동작 검증, 반복 작업 자동화에 적합합니다. 모든 명령(클릭·입력·스크린샷)은 **Java 리플렉션**으로 동작해 단일 코드베이스로 지원되는 모든 버전에서 작동하며 버전별 패치가 필요 없습니다.
