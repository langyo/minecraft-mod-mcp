Test World Reference Files
=========================

This directory contains reference screenshots for CI screenshot comparison testing.

## Reference Screenshot Naming

```
ref_{mc_ver}_{loader}_{label}.png

Examples:
  ref_1.21.7_forge_01_ingame.png
  ref_1.12.2_forge_03_sign.png
```

## Labels

- `01_ingame` — Player standing in the superflat Redstone Ready world
- `02_inventory` — Inventory open showing hotbar items
- `03_sign` — Sign placed with "Hello World" text

## How to Generate Reference Screenshots

1. Run a known-good version of MC with the mod installed
2. Use the `scripts/ci_helper.py` screenshot-test mode
3. Copy screenshots from `screenshots/ci/` to this directory
4. Rename them following the naming convention

## World Save

The test world is a Redstone Ready superflat world generated by
`scripts/ci_helper.py` via the `_generate_level_dat()` function.

World settings for deterministic renders:
- Game mode: Creative
- Time: Day (1000)
- Weather: Clear
- Difficulty: Normal
- Game rules: doFireTick=false, doMobSpawning=false, keepInventory=true
- options.txt: clouds=off, fov=70, renderDistance=4, particles=off, sounds=off
