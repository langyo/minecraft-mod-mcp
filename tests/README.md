## Reference Screenshot Naming

```
reference-screenshots/ref_{mc_ver}_{loader}.png           (screenshot-test job)
reference-screenshots/ref_{mc_ver}_{loader}_{label}.png   (e2e-smoke job)

Labels for e2e:
  - 01_ingame    — Player standing in the superflat Redstone Ready world
  - 02_inventory — Inventory open showing hotbar items
  - 03_sign      — Sign placed with "Hello World" text
```

## How to Generate Reference Screenshots

1. Run a known-good version of MC with the mod installed
2. Use `scripts/ci_helper.py screenshot-test` or `e2e` mode
3. Download screenshots from the CI `screenshots-*` artifact
4. Place validated screenshots in `tests/reference-screenshots/`

## Deterministic Render Settings

Generated by `scripts/ci_helper.py` → `generate_options_txt()`:
- clouds=off, particles=off, viewBobbing=false, entityShadows=false
- FOV=70, renderDistance=4, graphics=fast, brightness=100
- All sounds=off, showAutosaveIndicator=false

World generated by `_generate_level_dat()`:
- Redstone Ready superflat, Creative mode, Day time
- doFireTick=false, doMobSpawning=false, keepInventory=true
